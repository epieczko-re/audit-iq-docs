[[sect-exam-ready-bundle]]
== Exam-Ready Bundle Specification

=== Purpose

*Exam-ready bundle* = A complete, portable artifact of an audit suitable for export to examiners, regulators, or auditors.
Includes all evidence, findings, trail, and metadata.
Format: ZIP (structured directories) + optional merged PDF.

=== Detailed File Specifications

==== A. Audit Summary Report (PDF)

*Template structure:*

....
[Cover Page]
- Organization name, audit period, pack name, date
- Auditor name & signature block

[Executive Summary]
- Scope (record count, sampling methodology)
- Overall compliance assessment (Pass / Fail / Pass w/ Observations)
- Key findings summary by severity
  * Critical: N findings
  * Major: N findings
  * Minor: N findings
- Recommended corrective actions (summary)

[Detailed Findings]
- Table: Finding ID | Rule | Severity | Confidence | Evidence Link | Narrative | Approver

[Audit Trail Summary]
- Key events (creation, approvals, re-reviews)
- Timestamp, actor, action

[Attestation]
- "This audit was performed using AuditIQ Audit Pack [version]."
- "Audit trail, evidence, and scoring methodology documented in bundle."
- Digital signature or manager approval stamp
....

==== B. Findings Register (CSV + PDF)

*CSV columns:*

....
finding_id,rule_id,rule_name,target_record_id,severity,confidence_extraction,confidence_mapping,confidence_rule,combined_confidence,evidence_ids,narrative,approver_id,approval_date,status
....

*PDF:* Same data, formatted as professional table.

==== C. Corrective Action Log (CSV)

*Columns:*

....
finding_id,corrective_action_description,responsible_party,target_completion_date,evidence_reference_id,status
....

*Example:*

....
finding_001,"Implement missing disclosure language per Reg Z",mgmt_team_lead,2026-03-31,evidence_reg_z_template,open
....

==== G. Sampling Methodology (Markdown)

*Human-readable documentation:*

[source,markdown]
----
# Audit Sampling Methodology

## Population & Sample Design
- Population: 10,000 commercial loans originated in 2025
- Stratification: By loan type (term, revolving, HELOC)
- Sample size: 50 loans (0.5% of population)
- Confidence level: 95%
- Margin of error: ±5%

## Selection Method
Random stratified sampling using [method].

## Representativeness
Sample strata distribution matches population distribution:
- Term loans: 60% (30 in sample)
- Revolving: 30% (15 in sample)
- HELOC: 10% (5 in sample)

## Results
Zero critical findings; 2 minor findings (4% error rate).
----

=== Export Formats

*ZIP Bundle Structure:*

....
bundle_20260215_loanXYZ_hmda_v2p1.zip
├── bundle_manifest.json
├── audit_summary_report.pdf
├── findings_register.csv
├── findings_register.pdf
├── corrective_action_log.csv
├── audit_trail_excerpt.json
├── source_document_index.json
├── pack_configuration.json
├── sampling_methodology.md
└── reproducibility_attestation.json
....

*Optional: Merged PDF* - Single PDF combining audit summary + findings + attestation - Use case: Email-friendly; print-friendly - Metadata embedded in PDF (searchable)
