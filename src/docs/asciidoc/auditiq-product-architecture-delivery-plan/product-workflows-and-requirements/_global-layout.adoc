=== Global Layout: Shared Navigation Shell

All screens share a consistent layout shell: left sidebar navigation, top bar with breadcrumbs and user menu, and a scrollable main content area.
This follows the Flowbite Dashboard Layout pattern.

*Flowbite components:* Sidebar, Navbar, Breadcrumb, Avatar, Dropdown.

*Layout specifications:*

* Sidebar width: 256px (collapsible to 64px icon-only mode on mobile)
* Top bar height: 64px with breadcrumb trail and user avatar dropdown
* Main content area: fluid width, max-width 1440px, centered
* Responsive breakpoints: mobile (<768px sidebar collapses), tablet (768–1024px), desktop (>1024px)
