=== Stage 5: Dashboards

Dashboards provide read-only operational and executive views across all audits.

==== User Journey

*Primary workflow alignment:* supports closed-loop visibility for <<user-journey-overview, Workflow 1>> and execution-mode workflows (5-7).
*Execution modes:* Read-only views across all modes.

*Screen progression in this stage:*

. <<wf-exec-dashboard, WF-19 Executive Dashboard>> — portfolio-level health and trend monitoring.
. <<wf-mgr-dashboard, WF-20 Manager Dashboard>> — operational workload, SLAs, and pending approvals.
. <<wf-drift-monitor, WF-21 Drift Monitoring>> — model and quality drift signals (Phase 2).

*Key interactions:*

* *[Download Quarterly Report]* generates executive summary PDF
* All data scoped to user's tenant (multi-tenant isolation via PostgreSQL RLS)

[[wf-mgr-dashboard]]
===== WF-20: Manager Dashboard

*Purpose:* Operational view for audit managers.
Shows active audits, critical findings requiring attention, overdue remediations, and team workload.

*Roles:* Manager, Audit Lead.

*Flowbite components:* Card, Table, Badge, Alert, Chart, Button.

[plantuml]
----
@startsalt
{+
  <b>Manager Dashboard</b>
  { Home > Dashboard (Manager) }
  ===
  <color:Red><b>! 2 Critical findings untriaged > SLA | 1 remediation overdue</b></color>
  ===
  {
    {+
      <b>My Active Audits</b>
      {#
        <b>Audit</b> | <b>Status</b> | <b>Progress</b> | <b>Findings</b> | <b>Action</b>
        HMDA Q1 2026 | <color:RoyalBlue>In Progress</color> | 67% | 23 (2 Crit) | [View]
        CRA 2026 | <color:RoyalBlue>In Progress</color> | 35% | 8 (0 Crit) | [View]
        TRID Q1 2026 | <color:Orange>Review</color> | 100% | 15 (1 Crit) | [Finalize]
      }
    }
    |
    {+
      <b>Team Workload</b>
      {#
        <b>Reviewer</b> | <b>Open Findings</b> | <b>Overdue</b>

      }
    }
  }
  ---
  {+
    <b>Critical Findings Requiring Action</b>

    }
  }
  ---

    {#
      <b>Type</b> | <b>Item</b> | <b>Requested By</b> | <b>Action</b>
      AI Narrative | F-001 narrative (HMDA Q1) | System | [Review & Approve]
      Finding Closure | F-005 (HMDA Q1) | Jane Smith | [Approve Closure]
      Report | HMDA Q1 Audit Report | Jane Smith | [Review Report]
    }
  }
  ---
  { [View Aging Dashboard] | [View All Findings] | [Generate Report] }
}
@endsalt
----

*Key interactions:*

* *Alert banner* shows immediate-action items (SLA violations, overdue remediations)
* *Critical Findings* table shows only findings requiring manager attention; sorted by urgency
* *Pending Approvals* consolidates all items requiring manager sign-off: AI narratives, finding closures, reports
* *Team Workload* card helps balance assignments across reviewers
* *[Triage Now]* navigates to <<wf-finding-detail>> with triage section pre-expanded
